IDEA集成Git

IDEA版本2021.3，不同版本的按钮位置和按钮内容大体一致。

(1)打开IDEA的设置（setting）

编辑

(2)点开版本控制中的Git，选择文件路径。文件路径为安装的Git目录下的bin目录下的git.exe。

编辑 编辑

（3）选择完成后点击测试，会显示Git版本信息，然后点击确定。

编辑 （4）此时右键点击项目根目录会出现Git选项，此时已经集成成功。点击添加则添加到Git暂存区，点击提交目录即提交到本地库。

查看版本控制信息

切换版本

编辑

新建分支

点击右下角的分支管理。

编辑 点击新分支即可创建分支。

编辑
